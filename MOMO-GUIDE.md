# 💖 墨墨的魔法单词王国 - 使用指南

## 亲爱的墨墨：

这是一个专门为你打造的魔法单词学习游戏！在这里，你可以通过有趣的游戏来学习英语单词，每天都会有新的进步哦～

## 🎮 游戏特色

### 1. 墨墨的专属欢迎

- 每次打开游戏，都会看到"墨墨，欢迎回来！"的问候
- 顶部有"墨墨的学习之旅"，记录你的成长
- 游戏中所有鼓励都是专门为墨墨准备的

### 2. 三种有趣的游戏模式

#### 🎯 配对游戏
找到单词和emoji图标的正确配对。比如看到🐱，就要找到"cat"！

**墨墨会收到的鼓励**：
- "墨墨真棒！"
- "墨墨好厉害！"
- "太棒了墨墨！"

#### ✏️ 拼写游戏
看到emoji图标，用键盘输入对应的英文单词。

**提示**：不用记中文意思，看图直接拼写会让墨墨的英语思维更好！

#### 🎲 选择题
看到图片，从4个选项中选出正确的单词。

### 3. 艾宾浩斯记忆系统

游戏会自动记住墨墨学过的每一个单词，并在最合适的时间提醒墨墨复习：

| 复习时间 | 说明 |
|---------|------|
| 5分钟后 | 刚学完立刻复习，加深印象 |
| 1天后 | 第一次间隔复习 |
| 2天后 | 第二次间隔复习 |
| 7天后 | 一周后巩固 |
| 15天后 | 半月后强化 |
| 30天后 | 一个月后确认精通 |

### 4. 墨墨的掌握等级

- ⚪ **新单词**：第一次见到
- 🟡 **学习中**：正在学习，还不太熟
- 🔵 **熟悉**：已经认识了，很不错！
- 🟢 **精通**：完全掌握，太棒了墨墨！

## 🌟 给墨墨的建议

### 每天学习计划

1. **早上起床后**（5-10分钟）
   - 玩一局配对游戏，唤醒大脑
   - 看看有没有需要复习的单词

2. **放学回家后**（10-15分钟）
   - 学习5-10个新单词
   - 完成拼写游戏或选择题

3. **睡觉前**（5分钟）
   - 快速复习今天学的单词
   - 看看自己的进步

### 学习小技巧

1. **不要怕错**：每次答错，游戏都会记录下来，之后会重点帮墨墨练习这个单词
2. **每天坚持**：哪怕只学5个单词，坚持下去就会看到神奇的效果
3. **关注提醒**：看到"有单词需要复习"的红色提醒时，一定要玩一局哦
4. **争取三星**：每一关都努力得三颗星⭐⭐⭐
