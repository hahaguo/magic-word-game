# 离线使用和进度备份功能说明

## 📱 PWA离线支持

### 什么是PWA？
PWA (Progressive Web App) 是一种可以像原生应用一样使用的网页应用。安装后可以：
- 📲 添加到手机/平板桌面，像App一样打开
- ✈️ 完全离线使用，无需网络连接
- ⚡ 加载速度更快
- 🔄 自动更新到最新版本

### 如何安装PWA？

#### iOS (iPhone/iPad)
1. 使用Safari浏览器打开应用
2. 点击底部分享按钮 📤
3. 选择"添加到主屏幕"
4. 点击"添加"
5. 在桌面找到"魔法单词"图标即可使用

#### Android
1. 使用Chrome浏览器打开应用
2. 点击右上角菜单 ⋮
3. 选择"添加到主屏幕"或"安装应用"
4. 点击"安装"
5. 在桌面找到应用图标

#### 桌面浏览器 (Chrome/Edge)
1. 打开应用
2. 地址栏右侧会出现安装图标 ⊕
3. 点击"安装"
4. 应用会作为独立窗口打开

### 离线功能

应用会自动缓存所有必要文件：
- ✅ 所有单词数据
- ✅ 游戏逻辑和界面
- ✅ 学习记录（保存在本地）
- ✅ 完整的游戏功能

**无需网络也能正常学习！**

## 💾 数据备份功能

### 为什么需要备份？
- 📱 换设备时不丢失学习进度
- 🔒 防止意外数据丢失
- 👨‍👩‍👧‍👦 在多个设备间同步学习进度
- 💿 长期保存学习记录

### 如何备份数据？

1. **导出数据**
   - 点击主页的"📊 查看详细统计"
   - 滚动到底部的"💾 数据备份与恢复"区域
   - 点击"📥 导出学习数据"按钮
   - 数据会自动下载为JSON文件
   - 文件名格式：`墨墨的学习数据_2024-01-01.json`

2. **保存备份文件**
   - 📧 发送到邮箱保存
   - ☁️ 上传到云盘（百度网盘、iCloud等）
   - 💾 复制到U盘
   - 📱 通过微信/QQ发送给自己

### 如何恢复数据？

1. **导入数据**
   - 在新设备上打开应用
   - 点击"📊 查看详细统计"
   - 点击"📤 导入学习数据"按钮
   - 选择之前导出的JSON文件
   - 确认导入

2. **注意事项**
   - ⚠️ 导入会**覆盖**当前所有数据
   - 💡 建议先导出当前数据作为备份
   - ✅ 导入成功后会看到确认提示

### 导出文件包含什么？

```json
{
  "version": "2.3",
  "exportDate": "2024-01-01T12:00:00.000Z",
  "totalWords": 50,
  "memoryData": {
    // 每个单词的学习记录
    "apple": {
      "firstStudyTime": 时间戳,
      "lastStudyTime": 时间戳,
      "nextReviewTime": 时间戳,
      "reviewCount": 复习次数,
      "correctCount": 正确次数,
      "wrongCount": 错误次数,
      "masteryLevel": 掌握等级,
      "reviewHistory": [历史记录]
    }
  }
}
```

## 🗑️ 清除数据

如果需要重新开始学习：
1. 点击"🗑️ 清除所有数据"按钮
2. **会有两次确认**，防止误操作
3. 建议清除前先导出备份

## 📊 数据统计

备份文件中包含：
- 已学习的单词数量
- 每个单词的复习历史
- 正确率统计
- 掌握程度等级
- 下次复习时间

## 🔐 数据安全

- 所有数据保存在浏览器本地
- 不会上传到任何服务器
- 导出的文件完全由用户控制
- 可以随时删除或备份

## 💡 使用建议

1. **定期备份**
   - 建议每周备份一次
   - 学习100个单词后备份
   - 换设备前一定要备份

2. **多设备学习**
   - 在手机上导出数据
   - 在平板上导入数据
   - 实现学习进度同步

3. **长期保存**
   - 将备份文件上传云盘
   - 作为学习历程的记录
   - 方便将来查看学习轨迹

## ❓ 常见问题

**Q: 离线后能使用所有功能吗？**
A: 是的！除了需要下载新内容，所有游戏、复习、统计功能都能离线使用。

**Q: 数据会丢失吗？**
A: 数据保存在浏览器，除非清除浏览器数据或卸载应用。建议定期备份。

**Q: 可以在不同浏览器间同步吗？**
A: 需要手动导出/导入。每个浏览器的数据是独立的。

**Q: 备份文件安全吗？**
A: 文件只包含学习记录，没有个人信息，完全安全。

**Q: 可以编辑备份文件吗？**
A: 不建议手动编辑，可能导致数据损坏。如需修改，请在应用内操作。
