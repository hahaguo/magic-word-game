# 艾宾浩斯记忆曲线系统说明

## 什么是艾宾浩斯记忆曲线？

德国心理学家赫尔曼·艾宾浩斯（Hermann Ebbinghaus）通过研究发现，人的记忆会随着时间推移而遗忘，但通过科学的复习可以有效巩固记忆。

我们的游戏完全基于艾宾浩斯记忆曲线设计，让孩子的单词学习更加科学高效！

## 📊 记忆系统工作原理

### 1. 复习时间安排

根据记忆曲线，系统自动安排以下复习时间：

- **第1次复习**：学习后 5分钟
- **第2次复习**：学习后 1天
- **第3次复习**：学习后 2天
- **第4次复习**：学习后 7天
- **第5次复习**：学习后 15天
- **第6次复习**：学习后 30天

### 2. 智能单词推荐

每次开始游戏时，系统会智能选择单词：

- **70%** 需要复习的单词（根据复习时间到期）
- **30%** 新单词

这样既能巩固已学单词，又能学习新知识！

### 3. 单词掌握等级

系统根据学习表现自动评定单词掌握程度：

| 等级 | 说明 | 条件 |
|------|------|------|
| ⚪ 新单词 | 尚未学习 | 第一次见到 |
| 🟡 学习中 | 刚开始学习 | 复习1次以上，正确率一般 |
| 🔵 熟悉 | 比较熟悉 | 复习3次以上，正确率≥70% |
| 🟢 精通 | 完全掌握 | 复习6次以上，正确率≥90% |

### 4. 智能难度调整

- **答对**：推进到下一个复习间隔，强化记忆
- **答错**：回退到较短的复习间隔，重点练习

## 🎮 功能特点

### 学习追踪

- **实时记录**：每个单词的学习时间、复习次数
- **正确率统计**：记录每次答对/答错情况
- **复习提醒**：自动提示需要复习的单词数量

### 学习数据

系统自动保存以下数据到浏览器（localStorage）：

- ✅ 每个单词的详细学习记录
- ✅ 下次复习时间
- ✅ 掌握等级
- ✅ 历史学习记录
- ✅ 正确率和错误次数

### 学习统计

点击"查看详细统计"可以看到：

- 📖 **已学单词**：学习过的单词总数
- ⭐ **精通单词**：完全掌握的单词数
- 🎯 **正确率**：整体答题准确率
- 📚 **待复习**：需要复习的单词数量
- 📊 **掌握程度分布**：可视化展示各等级单词占比

### 今日学习目标

- 每天推荐学习 **20个单词**
- 实时显示今日进度
- 鼓励每天坚持学习

## 💡 使用建议

### 给家长的建议

1. **每天学习**：建议孩子每天学习15-20分钟
2. **关注复习提醒**：看到复习提醒时，优先复习旧单词
3. **查看统计数据**：定期检查学习进度和掌握情况
4. **鼓励坚持**：记忆曲线需要长期坚持才有效果

### 给孩子的建议

1. **认真答题**：每次答题都要认真思考
2. **答错不要紧**：系统会自动安排重点练习
3. **坚持复习**：看到复习提醒就玩一局游戏
4. **争取精通**：努力让更多单词达到"精通"等级

## 🔧 高级功能

### 数据管理

在浏览器控制台（F12）中可以使用以下命令：

```javascript
// 查看所有学习数据
console.log(memorySystem.memoryData);

// 查看统计信息
console.log(memorySystem.getStatistics());

// 查看需要复习的单词
console.log(memorySystem.getWordsToReview());

// 导出学习数据（备份）
memorySystem.exportData();

// 清除所有数据（重新开始）
memorySystem.clearAllData();
```

### 数据持久化

- 学习数据自动保存在浏览器本地
- 即使关闭浏览器，数据也不会丢失
- 更换浏览器或设备时，可以导出/导入数据

## 🌟 科学依据

### 为什么有效？

1. **遗忘曲线规律**：人在学习后的20分钟、1小时、1天会迅速遗忘，艾宾浩斯记忆曲线正好在这些关键时间点安排复习

2. **间隔重复**：通过逐渐拉长的复习间隔，将短期记忆转化为长期记忆

3. **主动回忆**：游戏中需要主动回忆单词，比被动阅读效果好10倍

4. **即时反馈**：答对答错立即知道结果，强化正确记忆

5. **个性化学习**：根据每个单词的掌握情况调整复习频率，学习更高效

## 📈 预期效果

坚持使用本系统学习：

- **1周**：熟悉20-30个新单词
- **1个月**：掌握80-100个单词，其中20-30个达到精通
- **3个月**：掌握200+个单词，建立良好的英语学习习惯
- **6个月**：所有小学1-3年级单词基本精通

## 🎯 学习路径建议

### 第1阶段：适应期（1-2周）

- 每天学习10-15个单词
- 熟悉游戏玩法
- 建立学习习惯

### 第2阶段：提升期（3-8周）

- 每天学习15-20个单词
- 重视复习提醒
- 争取提高正确率

### 第3阶段：巩固期（2-4个月）

- 保持每天复习
- 提升精通单词数量
- 挑战难度更高的模式

## ✨ 特别提醒

1. **不要清理浏览器数据**：会导致学习记录丢失
2. **定期导出备份**：防止意外丢失数据
3. **多种游戏模式结合**：配对、拼写、选择题都玩，全方位记忆
4. **家长参与**：家长可以和孩子一起玩，提高学习兴趣
