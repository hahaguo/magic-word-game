## 🎯 关键技术点

### 1. 记忆曲线算法

```javascript
// 复习间隔（毫秒）
reviewIntervals = [
    5 * 60 * 1000,          // 5分钟
    24 * 60 * 60 * 1000,    // 1天
    2 * 24 * 60 * 60 * 1000,    // 2天
    7 * 24 * 60 * 60 * 1000,    // 7天
    15 * 24 * 60 * 60 * 1000,   // 15天
    30 * 24 * 60 * 60 * 1000    // 30天
]

// 答对：推进到下一个间隔
nextReviewTime = now + reviewIntervals[reviewCount]

// 答错：回退2个间隔
nextReviewTime = now + reviewIntervals[Math.max(0, reviewCount - 2)]
```

### 2. 掌握度判定

```javascript
if (reviewCount >= 6 && accuracy >= 0.9) {
    // 精通
} else if (reviewCount >= 3 && accuracy >= 0.7) {
    // 熟悉
} else if (reviewCount > 0) {
    // 学习中
} else {
    // 新单词
}
```

### 3. 数据持久化

```javascript
// 保存
localStorage.setItem('wordMemoryData', JSON.stringify(memoryData))

// 读取
memoryData = JSON.parse(localStorage.getItem('wordMemoryData'))
```

## 🎓 科学依据

本系统基于以下心理学原理：

1. **艾宾浩斯遗忘曲线**：记忆会随时间衰减，科学复习可对抗遗忘
2. **间隔重复**：逐渐拉长复习间隔，将短期记忆转化为长期记忆
3. **主动回忆**：游戏中的主动回忆比被动阅读效果好10倍
4. **即时反馈**：立即知道对错，强化正确记忆
5. **个性化学习**：根据个人掌握情况调整，学习更高效
